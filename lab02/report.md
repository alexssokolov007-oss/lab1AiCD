## Формулировка задания

Реализовать таблицу со структурой №1: целый ключ и данные хранятся вместе в одной записи.

Для `N = 19`:
- метод сортировки: `((N - 1) % 8) + 1 = 3` — пузырьковая сортировка;
- структура таблицы: `((N + 5) % 8) + 1 = 1`;
- минимальное количество элементов для структуры №1: 11.

Программа должна:
- читать таблицу из файла;
- сортировать таблицу методом пузырьковой сортировки;
- выполнять бинарный поиск по целому ключу;
- демонстрировать работу алгоритмов на трех случаях: уже упорядоченная таблица, таблица в обратном порядке, неупорядоченная таблица.

## Описание выполненной работы

### Структура проекта

Проект состоит из следующих файлов:

#### 1. `main.c` - Основная программа и реализация алгоритмов

Содержит структуру таблицы, функции работы с памятью, чтение данных из файла, пузырьковую сортировку и бинарный поиск.

Структура записи таблицы:

```c
typedef struct {
    int key;
    char value[MAX_VALUE_LEN];
} Record;
```

Структура таблицы:

```c
typedef struct {
    Record *items;
    int size;
} Table;
```

**Основные функции:**
- `create_table()` — создание таблицы заданного размера
- `free_table()` — освобождение памяти таблицы
- `copy_table()` — копирование таблицы
- `read_table_from_file()` — чтение таблицы из файла
- `print_table()` — вывод таблицы на экран
- `swap_records()` — обмен записей таблицы
- `bubble_sort()` — пузырьковая сортировка таблицы по ключу
- `binary_search()` — бинарный поиск по ключу
- `make_sorted_copy()` — создание упорядоченной копии через `qsort`
- `reverse_table()` — разворот таблицы
- `process_case()` — обработка одного случая: вывод, сортировка, интерактивный поиск

Реализация пузырьковой сортировки:

```c
void bubble_sort(Table *table) {
    for (int i = 0; i < table->size - 1; i++) {
        int swapped = 0;

        for (int j = 0; j < table->size - 1 - i; j++) {
            if (table->items[j].key > table->items[j + 1].key) {
                swap_records(&table->items[j], &table->items[j + 1]);
                swapped = 1;
            }
        }

        if (!swapped) {
            break;
        }
    }
}
```

Реализация бинарного поиска:

```c
int binary_search(const Table *table, int target_key) {
    int left = 0;
    int right = table->size - 1;

    while (left <= right) {
        int mid = left + (right - left) / 2;

        if (table->items[mid].key == target_key) {
            return mid;
        }

        if (table->items[mid].key < target_key) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }

    return -1;
}
```
