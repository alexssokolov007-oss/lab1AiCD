# Отчет по лабораторной работе 2

## Тема
Сортировка и поиск.

## Вариант
19:
- метод сортировки: пузырьковая сортировка (№3);
- структура таблицы: №1 (ключ целый, хранение вместе);
- минимальное число элементов: 11.

## Задание
Составить программу для сортировки таблицы и двоичного поиска по ключу.
Проверить сортировку в трех случаях: уже упорядочена, обратный порядок, неупорядочена.

## Реализация
- Сортировка: пузырьковая, с ранним выходом.
- Поиск: двоичный поиск в отсортированном массиве.
- Три сценария запускаются последовательно.
