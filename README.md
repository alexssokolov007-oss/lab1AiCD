# АиСД — лабораторные

Решения по вариантам хранятся в отдельных папках `lab01`..`lab10`.

Сборка:
- `make` (или `make lab01`, `make lab02`)
- или через CMake (`CMakeLists.txt`).

Тесты:
- `make test` (5 тестов на каждую лабу)
- скрипт: `tests/run_tests.ps1`

Отчеты оформляются в Markdown в каждой папке: `lab01/report.md`, `lab02/report.md`.
