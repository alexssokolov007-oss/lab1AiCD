# Отчет по лабораторной работе 1

## Тема
Динамические структуры данных. Обработка деревьев.

## Вариант
19 — определить число вершин двоичного дерева.

## Задание
Реализовать двоичное дерево поиска (BST) и операции:
- добавление узла;
- визуализация дерева с отступами по глубине;
- удаление узла с корректным освобождением памяти;
- вычисление значения функции по варианту (подсчет числа вершин).

## Реализация
- Дерево — BST с ключом `int`.
- Вставка по правилу BST.
- Удаление — стандартный случай (0/1/2 потомка), при 2 потомках замена на минимальный элемент правого поддерева.
- Подсчет числа вершин — рекурсивный обход.
